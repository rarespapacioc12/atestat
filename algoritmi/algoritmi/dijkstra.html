Algoritmul lui Dijkstra

Algoritmul lui Dijkstra este utilizat pentru a calcula cel mai scurt drum de la un vârf sursă la toate celelalte vârfuri dintr-un graf ponderat fără muchii de cost negativ.

Descriere

Se pleacă de la vârful sursă și se explorează succesiv vârfurile cele mai apropiate, actualizând costurile drumurilor parcurse. La fiecare pas alegem vârful nevizitat cu distanța minimă.

Pseudocod (variantă pătratică - O(n^2))

function Dijkstra(n, start, cost):
    inițializează distanțele tuturor vârfurilor cu infinit
    distanța[start] = 0
    marchează toate vârfurile ca nevizitate

    for i = 1 to n:
        alege vârful nevizitat cu distanța minimă
        marchează-l ca vizitat
        pentru fiecare vecin u al vârfului curent:
            dacă distanța curentă + costul către u < distanța[u]:
                actualizează distanța[u]

Pseudocod (variantă cvasiliniară - O((n + m) log n))

function DijkstraHeap(n, start, lista_adiacenta):
    inițializează distanțele cu infinit
    distanța[start] = 0
    creează o coadă de priorități (heap)
    adaugă (0, start) în heap

    while heap nu este gol:
        (dist, v) = extrage minim din heap
        procesează vârful
        pentru fiecare vecin (u, cost) al lui v:
            dacă distanța[v] + cost < distanța[u]:
                actualizează distanța[u]
                adaugă (distanța[u], u) în heap

Explicații

În varianta pătratică, la fiecare pas căutăm manual minimul dintre vârfurile nevizitate. În varianta cvasiliniară folosim o coadă de priorități (heap) pentru a obține rapid vârful cu distanța minimă.

Heap-ul reduce timpul de extragere a minimului și actualizarea distanțelor, astfel accelerând algoritmul pentru grafuri mari și dense.

Exemplu de aplicare

Graful:

A -1- B
|    |
4    2
|    |
C -1- D

Pornind din A:

  • Distanța la A = 0
  • Distanța la B = 1
  • Distanța la C = 4
  • Distanța la D = 3

Complexitate

  • Complexitate timp (vector simplu): O(n^2)
  • Complexitate timp (heap): O((n + m) log n)
  • Complexitate spațiu: O(n + m)
