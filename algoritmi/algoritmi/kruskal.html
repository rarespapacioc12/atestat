Algoritmul lui Kruskal

Algoritmul lui Kruskal este folosit pentru a construi arborele de acoperire minimă (MST) al unui graf ponderat conex. Ideea de bază este alegerea muchiilor de cost minim care nu formează cicluri.

Descriere

Se sortează toate muchiile grafului în funcție de cost, apoi se adaugă în MST doar muchiile care unesc componente disjuncte, folosind DSU (Disjoint Set Union).

Pseudocod Kruskal

function Kruskal(n, lista_muchii):
    initializează DSU(n)
    sortează lista_muchii crescător după cost
    MST = []

    pentru fiecare muchie (u, v, cost):
        dacă Find(u) != Find(v):
            adaugă muchia în MST
            Union(u, v)

Funcțiile DSU folosite

Inițializare

function Initialize(n):
    pentru fiecare vârf i:
        parinte[i] = i
        rang[i] = 0

Find cu compresie de drum

function Find(x):
    dacă parinte[x] != x:
        parinte[x] = Find(parinte[x])
    return parinte[x]

Union după rang

function Union(x, y):
    rx = Find(x)
    ry = Find(y)
    dacă rx == ry:
        return
    dacă rang[rx] < rang[ry]:
        parinte[rx] = ry
    altfel dacă rang[rx] > rang[ry]:
        parinte[ry] = rx
    altfel:
        parinte[ry] = rx
        rang[rx] += 1

Explicații

Se sortează toate muchiile și se procesează în ordine crescătoare a costului. Se adaugă doar acele muchii care unesc două componente diferite (detectate cu DSU). Astfel evităm formarea ciclurilor.

Exemplu de aplicare

Lista muchiilor:

(A, B, 1)
(A, C, 4)
(B, C, 2)
(B, D, 5)
(C, D, 3)

Muchiile selectate vor fi A-B, B-C, C-D, cost total = 1+2+3=6.

Complexitate

  • Sortare muchii: O(m log m)
  • Operații DSU: aproape O(1) amortizat
  • Complexitate totală: O(m log m)
