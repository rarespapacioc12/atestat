Algoritmul Roy-Floyd

Algoritmul Roy-Floyd, cunoscut și ca Floyd-Warshall, este utilizat pentru a determina drumurile minime între toate perechile de vârfuri într-un graf ponderat (cu sau fără cicluri negative, dar fără cicluri negative de cost total negativ).

Descriere

Se pornește de la o matrice a costurilor (matricea de adiacență, unde cost[i][j] este costul arcului de la i la j, sau infinit dacă nu există arc). Se încearcă îmbunătățirea drumurilor existente trecând prin fiecare vârf intermediar.

Pseudocod

function RoyFloyd(n, cost):
    for k = 1 to n:
        for i = 1 to n:
            for j = 1 to n:
                if cost[i][k] + cost[k][j] < cost[i][j]:
                    cost[i][j] = cost[i][k] + cost[k][j]

Explicații

Se consideră toate tripletele (i, j, k) și verificăm dacă drumul de la i la j prin k este mai scurt decât drumul direct de la i la j. Dacă da, actualizăm costul.

Ordinea buclelor este importantă: bucla externă peste k asigură că vârful k este deja luat în calcul înainte de a procesa următoarele combinații.

Exemplu de aplicare

Având matricea inițială:

0   3 INF
 2   0   5
 INF INF 0

Apelăm algoritmul și după aplicare vom avea matricea:

0   3   8
 2   0   5
 INF INF 0

Complexitate

  • Complexitate timp: O(n^3)
  • Complexitate spațiu: O(n^2)
